# LukeUtilsMaster
老胡整理的工具类，后面会不断更新，自己在项目里边使用

使用方法
在工程build.gradle加入

	allprojects {
		repositories {

			maven { url 'https://jitpack.io' }
		}
	}

在项目build.gradle加入
dependencies {

	        compile 'com.github.huhaiAndroid:LukeUtilsMaster:1.1'

	}
